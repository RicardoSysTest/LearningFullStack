# Inicializar Proyecto

1. Crear una carpeta donde vamos a guardar el proyecto con el comando `mkdir api`
2. Entrar a la carpeta con el comando `cd api`
3. Para tener la direcion actual de trabajo usamos el comando `pwd`
4. Inicializamos nuestro proyecto de node con el comando `npm init -y`. Nota al agregar `-y` lo que hacemos a aceptar todas la opciones que vienen por defecto. Si queremos seleccionar cada opcion escribir el comando sin el `-y`. Al ejecutar el programa se debera de crar el archivo `package.json`. Editaremos el paquete json de nuestro proyecto cambiando el author a `Ricardo Yepez`
5. Posteriormente necesitamos crear otro archivo llamado `api.js`
6. Vamos a instalar unas dependencias de nodes con el comando `npm install -S express` 
7. Una vez hemos instalado `express` se ha va a crar un archivo `package-lock.json` Aeui se va guardar las versiones de los que paquetes que instalamos.
